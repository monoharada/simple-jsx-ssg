# simple-jsx-template-engin

To install dev dependencies:

```bash
bun install
```

To run:

```bash
bun run dev
```

To Build for Production

```bash
bun run build
```

To linting

```bash
bun lint
```

file structure

```tree
src
├── component
│ └── Image.tsx
│ └── SomeComponent.tsx
├── data
```
